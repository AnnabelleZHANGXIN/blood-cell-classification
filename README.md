# blood-cell-classification
Deep learning project for blood cell image classification with model evaluation and error analysis

## Overview
This project focuses on multi-class classification of blood cell images using deep learning models.

## Methods
- Models: CNN, ResNet-based, CNN-ViT hybrid
- Preprocessing: centre cropping, data augmentation
- Evaluation: validation accuracy, error analysis

## Key Results
- Achieved ~97% validation accuracy
- Improved performance through systematic hyperparameter tuning
- Identified misclassification patterns and potential overfitting issues

## Model Evaluation Insights
- Evaluated model generalisation across different data representations
- Analysed potential overfitting and data leakage risks
- Compared performance across architectures

## Tools
Python, PyTorch, Scikit-learn
